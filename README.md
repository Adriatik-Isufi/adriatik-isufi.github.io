# Fahrschule 06 – Deine Fahrschule für Aargau & Olten

Willkommen bei Fahrschule 06, der persönlichen Fahrschule von Vaxhid Mustafa. Wir begleiten dich Schritt für Schritt zum Führerschein – mit modernen Fahrzeugen, klaren Lernplänen und viel Geduld. Unser digitales Schaufenster findest du unter [https://www.fahrschule06.ch](https://www.fahrschule06.ch).

## Fahrschule 06 auf einen Blick

- Kategorie B (Auto) & Kategorie B BPT 121 (Taxi/berufsmässiger Personentransport)
- Kontrollfahrten für ausländische Führerausweise und Auffrischungsfahrten
- VKU, Nothelferkurs-Empfehlungen und Theoriecoaching
- Flexible Termine: früh morgens, abends und an Wochenenden
- Persönlicher Fahrlehrer: Vaxhid Mustafa, bekannt für ruhiges Coaching und klare Feedbacks

## Technische Basis (Kurzinfo)

- Framework: Next.js App Router + Tailwind CSS (siehe `app/page.tsx`).
- SEO: `app/layout.tsx` enthält Metadaten, OpenGraph und DrivingSchool-Structured-Data.
- Assets: Logos, Videos und Bilder liegen in `public/`, statische Success-Fotos unter `public/Success/`.
- Deployment: `pnpm build && pnpm start` oder Docker-Setup (`Dockerfile`, `docker-compose.yml`).
