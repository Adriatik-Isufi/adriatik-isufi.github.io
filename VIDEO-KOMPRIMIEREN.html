🎬 Neues Video komprimieren

Diese Anleitung funktioniert auf jedem Windows-PC – es muss nichts installiert sein. Alles Nötige (ffmpeg) wird beim ersten Mal automatisch heruntergeladen.

Schritt 1: Das neue Video in den public-Ordner des Projekts legen.
Schritt 2: Unten im Befehl NUR den Dateinamen in der ersten Zeile anpassen.
Schritt 3: Auf „Befehl kopieren" klicken.
Schritt 4: Windows-Taste drücken, powershell tippen, Enter. Dann im blauen Fenster: Rechtsklick (= einfügen) und Enter drücken.
Schritt 5: Warten bis „Fertig!" erscheint (1–3 Minuten). Danach wie gewohnt committen und pushen.
$video = "video1.mp4"   # <-- NUR DIESEN DATEINAMEN ANPASSEN

cd "$env:USERPROFILE\Documents\Projects\adriatik-isufi.github.io"
[Net.ServicePointManager]::SecurityProtocol = [Net.SecurityProtocolType]::Tls12
$ProgressPreference = "SilentlyContinue"
$ffdir = "$env:LOCALAPPDATA\ffmpeg-portable"
$ffmpeg = Get-ChildItem $ffdir -Recurse -Filter ffmpeg.exe -ErrorAction SilentlyContinue | Select-Object -First 1 -ExpandProperty FullName
if (-not $ffmpeg) {
  Write-Output "Lade ffmpeg herunter (einmalig, ca. 170 MB)..."
  New-Item -ItemType Directory -Force $ffdir | Out-Null
  Invoke-WebRequest -UseBasicParsing -Uri "https://github.com/BtbN/FFmpeg-Builds/releases/download/latest/ffmpeg-master-latest-win64-gpl.zip" -OutFile "$ffdir\ffmpeg.zip"
  Expand-Archive -Path "$ffdir\ffmpeg.zip" -DestinationPath $ffdir -Force
  Remove-Item "$ffdir\ffmpeg.zip"
  $ffmpeg = Get-ChildItem $ffdir -Recurse -Filter ffmpeg.exe | Select-Object -First 1 -ExpandProperty FullName
}
New-Item -ItemType Directory -Force "public\_originals" | Out-Null
if (Test-Path "public\_originals\$video") {
  Write-Output "Hinweis: Backup existiert schon - das Original wird NICHT ueberschrieben."
} else {
  Copy-Item "public\$video" "public\_originals\$video"
}
Write-Output "Komprimiere $video ... (1-3 Minuten)"
$tmp = "public\_originals\TEMP_$video"
& $ffmpeg -y -v error -i "public\$video" -c:v libx264 -crf 28 -preset slow -profile:v high -pix_fmt yuv420p -c:a aac -b:a 96k -movflags +faststart $tmp
if ($LASTEXITCODE -ne 0 -or -not (Test-Path $tmp)) {
  Write-Output "FEHLER: Komprimierung fehlgeschlagen! Das Original wurde NICHT veraendert. Bitte Dateinamen pruefen."
  if (Test-Path $tmp) { Remove-Item $tmp }
} else {
  Move-Item -Force $tmp "public\$video"
  "VORHER:  {0:N1} MB" -f ((Get-Item "public\_originals\$video").Length / 1MB)
  "NACHHER: {0:N1} MB" -f ((Get-Item "public\$video").Length / 1MB)
  Write-Output "Fertig! Das unkomprimierte Original liegt als Backup in public\_originals\ (wird nicht committet)."
}
✓ Kopiert!

Qualität: CRF 28 + preset slow – bei Handy-Videos visuell nicht vom Original zu unterscheiden (wurde per Einzelbild-Vergleich geprüft). Das Original bleibt als Backup in public\_originals\ erhalten (dieser Ordner ist git-ignoriert und landet nie auf der Website).

Falls das Video danach komisch aussieht (unwahrscheinlich): einfach das Backup aus public\_originals\ zurückkopieren.
